# Smart Finance Dashboard 

A modern and responsive finance dashboard UI built with HTML and CSS.

This project provides a clean interface for managing personal finances, tracking transactions, monitoring budgets, and viewing financial insights.

## ✨ Features

- 📊 Balance overview cards
- 💳 Payment methods section
- 📑 Recent transactions table
- 📈 Income & expenses chart UI
- 💰 Budget overview tracking
- 🧾 Monthly bills management
- 👤 Account summary section
- 📱 Fully responsive design
- 🎨 Modern dashboard UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- Font Awesome Icons
- Responsive Design (Media Queries)
